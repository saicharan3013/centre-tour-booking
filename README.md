# centre-tour-booking# Centre Tour Booking Calendar

A responsive front-end prototype built using React and Vite for preschool centre tour scheduling.

## Features

* Home Page
* Centre Tour Booking Form
* Parent Information Collection
* Date and Time Slot Selection
* Booking Confirmation Page
* Dashboard for Viewing Bookings
* Local Storage Integration
* Responsive Design

## Tech Stack

* React
* Vite
* React Router DOM
* CSS

## Installation

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
